Aide – Gestion des tournois

Guide complet pour organiser, gérer et participer aux tournois sur HubISoccer

1. Créer un tournoi

1

Choisir le type de tournoi

Sur la page d'accueil du module, cliquez sur "Créer un tournoi". Sélectionnez le type :
- Public Show You : accessible à tous (code d'accès public).
- Public Détection : réservé aux joueurs ayant validé le premier‑pas.
- Privé HubISoccer : pour les membres de l'espace privé.
- Privé Simple : nécessite un paiement forfaitaire (pour les organisateurs externes).

2

Remplir les informations générales

Nom du tournoi, sport, dates, lieu, description, montant de la prime éventuelle, code d'accès (pour les publics). Vous pouvez également ajouter un flux vidéo (YouTube, Twitch) pour la retransmission.

3

Définir le règlement

Ajoutez un règlement spécifique (facultatif). Pour les tournois privés simples, un règlement est obligatoire et doit être accepté par les participants avant inscription.

4

Valider la création

Le tournoi apparaît alors dans la liste des tournois. Les inscriptions sont ouvertes selon les modalités choisies.

2. Gérer les inscriptions

Depuis votre tableau de bord ("Mes tournois" puis "Gérer"), vous pouvez :

  • Visualiser la liste des demandes d'inscription (en attente).
  • Approuver ou rejeter chaque demande.
  • Inviter directement des équipes ou des joueurs (onglet "Inviter").
  • Exporter la liste des participants (CSV).
Astuce : Pour les tournois publics avec code d'accès, assurez-vous de communiquer le code aux personnes concernées. Pour les tournois privés simples, le paiement du forfait est requis avant validation.

3. Organiser les matchs

Une fois les inscriptions validées, vous pouvez créer les rencontres :

  1. Dans la gestion du tournoi, onglet "Matchs" → "Ajouter un match".
  2. Sélectionnez les deux équipes, la date, le lieu, et éventuellement l'URL du streaming.
  3. Les matchs apparaissent dans le calendrier du tournoi et dans la page détail.
  4. Pendant le match, vous (ou les officiels désignés) pouvez saisir les événements (buts, cartons, etc.) depuis l'onglet "Rapports".

Les statistiques sont automatiquement mises à jour à partir des événements saisis.

4. Saisir les rapports de match

Chaque match peut être accompagné de trois types de rapports :

  • Rapport arbitre : score, cartons, incidents.
  • Rapport commissaire : affluence, organisation, observations.
  • Rapport médical : blessures, interventions.

Pour accéder au formulaire, cliquez sur "Saisir rapport" dans les détails du match. Les rapports sont ensuite consultables et exportables (PDF, Excel).

5. Suivre les classements et statistiques

Les classements sont calculés automatiquement à partir des résultats et sont visibles dans l'onglet "Classement" de la page du tournoi. Vous y trouverez :

  • Classement général des équipes (points, victoires, etc.)
  • Meilleurs buteurs, passeurs
  • Statistiques par joueur (cartons, matchs joués)

Les statistiques individuelles alimentent le profil des joueurs sur la plateforme.

6. Gestion des primes

Si votre tournoi prévoit une prime, vous pouvez l'attribuer à une équipe ou à un joueur dans l'onglet "Primes" de la gestion du tournoi. Les primes peuvent être versées via la plateforme (intégration à venir) ou en direct selon les modalités définies.

7. Foire aux questions (FAQ)

Comment modifier les informations d'un tournoi après sa création ?
Rendez-vous dans "Mes tournois", cliquez sur "Gérer" puis sur l'onglet "Informations". Vous pourrez y modifier le nom, les dates, le lieu, la prime, etc.
Que faire si une équipe se désiste en cours de tournoi ?
Vous pouvez déclarer le forfait dans la gestion des matchs concernés. Le classement sera ajusté automatiquement. Pour les tournois avec primes, l'équipe forfaitaire peut perdre ses droits.
Puis-je ajouter un flux vidéo après la création du tournoi ?
Oui, modifiez les informations du tournoi et renseignez l'URL du flux. Celui-ci sera alors visible dans la page du tournoi et dans la page dédiée "Live Stream".
Comment exporter les résultats du tournoi ?
Dans la page du tournoi, un bouton "Exporter" (en cours de développement) permettra de télécharger les classements, les statistiques et les rapports de match au format CSV ou Excel.

8. Besoin d'aide supplémentaire ?

Si vous rencontrez un problème technique ou une question non traitée ici, vous pouvez :

  • Contacter le support via le formulaire dans "Support" (accessible depuis la navbar).
  • Consulter le Règlement général pour les aspects juridiques.
  • Rejoindre la communauté sur le "Feed" pour échanger avec d'autres organisateurs.
